# Wumpus World

پروژهٔ دانشگاهی برای پیاده‌سازی و مقایسهٔ سه روش هوش مصنوعی در بازی Wumpus World.

## وضعیت فعلی

مرحلهٔ ۱: اسکلت Python و smoke test. هنوز موتور بازی یا عامل‌ها پیاده‌سازی نشده‌اند.

## نصب توسعه

```bash
python -m venv .venv
source .venv/bin/activate       # Windows: .venv\Scripts\activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
```

## اجرای تست

```bash
python -m pytest
```
